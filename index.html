Gmailemojigenerator

Emoticon Art Generator for GMail. Allows the conversion of a pixelated image into Emoji Art that can be used in GMail emails.

download .ZIPdownload .TGZ

Version: 0.01

Script to generate Emotji/Emoticon Art for GMail using a source image. Ideally the source image should be a pixelated image but the script will do its best to pixelate an image before creating the art.

Usage: python EmojiGenerator.py ImagePath NumOfColumns NumOfRows Threshold(optional).
